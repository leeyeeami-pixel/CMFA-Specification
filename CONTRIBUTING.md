# Contributing to CMFA

Welcome to CMFA-WDG.

## Ways to Contribute

- Submit protocol improvement proposals
- Define new Brick Classes
- Add compliance tests
- Provide reference hardware drafts

## Contribution Workflow

1. Fork this repository
2. Create a feature branch
3. Submit a Pull Request
4. Discussion occurs via Issues
5. Draft revisions are merged by consensus

## Naming Rules

All specifications must follow:

CMFA-XXXX_Title.md

Example:

CMFA-3003_Extension_Headers.md
